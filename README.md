# PetPoint
A simple and reliable Python-based scheduling system designed to help pet groomers efficiently organize appointments, clients, and services.

# Overview:

PetPoint is a Python program designed to help pet groomers manage their appointments more efficiently. Users can add, view, and search for appointments, with all information saved in a simple text file for easy access and organization.

## Features
- Add new grooming appointments with pet and owner details  

- View a list of upcoming appointments  

- Search by pet name or owner name  

- Save appointment data to a file for future access 


## Target Users: 
Small grooming businesses, salon owners, and mobile pet groomers who want an organized way to track daily appointments.
